# x-ui

这是一个探索前端工程化和组件库的项目，还比较粗糙和简陋

使用 typescript + react + scss 编写组件，文档的搭建使用 dumi 

## 快速开始

安装

```shell
yarn add @windfall/x-ui
```
使用 

```tsx | pure
import { Button } from '@windfall/x-ui'

ReactDOM.render(<Button />, mountNode)
```

引入样式

```tsx | pure
import '@windfall/x-ui/dist/index.css'
```
